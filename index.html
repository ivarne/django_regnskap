A simple attemt to make an accounting application for django. Ugly mix of Norwegian and English in variable names and comments. You can freely use this code for whatever purpuse you'd like, but I take no responsibility.

Download as .zip Download as .tar.gz View on GitHub

List of dependencies: Installable using easy_install: django # of course, it's a django app openpyxl # to enable Excel export dropbox # for dropbox upload and make security copies of files mathplotlib # for making graphs

To install: copy local_settings.py.dist to local_settings.py and insert your local settings (Database, apiKeys...)

Then it should behave as a normal django app:)
